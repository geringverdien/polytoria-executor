# Build

- Visual Studio 2022
- Download xmake from [https://xmake.Io](https://xmake.io)

### Download

#### Build from source

```sh
git clone https://github.com/ElCapor/polytoria-executor.git

cd polytoria-executor

xmake

Open PolyHack.sln in visual studio and press build

```
